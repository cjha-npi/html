Detailed Description

Concept to check if two types are related through inheritance.

Checks whether two types are related through inheritance, either in a base-to-derived or derived-to-base direction. This is useful for verifying safe polymorphic usage such as upcasting or downcasting between types.

Example:
struct A {};
struct B : A {};
struct C {};
PolymorphicallyRelated<A, B> // True
PolymorphicallyRelated<B, A> // True
PolymorphicallyRelated<A, A> // True
PolymorphicallyRelated<A, C> // False
