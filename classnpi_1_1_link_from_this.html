Detailed Description

template<typename T>
class npi::LinkFromThis< T >

Enables objects that inherits from this class to produce a Link object referencing themselves if the object is wrapped in a LinkHost.

  • Behaves like a std::enable_shared_from_this but for LinkHost.
  • Stores a weak reference to the universal observer in the LinkHost.
  • Provides a way to obtain a Link object to itself using the universal observer if the LinkHost object is still alive.
Template Parameters
TThe type of the object being wrapped in LinkHost and supporting enable-from-this semantics.
Note
This class only holds a weak reference to the universal observer in the LinkHost. This means that as long as any LinkHost object is alive, the LinkFromThis object will be able to create a Link object to itself and that Link object will be valid as long as the LinkHost object is alive.
Example of using LinkFromThis:
struct MyClass : LinkFromThis<MyClass> {
void hello() {}
};
auto host = LinkHost<MyClass>(new MyClass());
auto link = host.universal_()->linkFromThis_();
if(link) link->hello();
