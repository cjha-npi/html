Detailed Description

template<typename T>
class npi::LinkHost< T >

A thread-safe, owning host that manages an object of type T and notifies linked pointers upon destruction.

  • Behaves like a std::shared_ptr i.e. owns the object T and destroys it when the last reference is released.
  • Provides direct pointer-like access to the object.
  • Supports polymorphic conversion and comparison.
  • Maintains two Link channels: Universal and Local.
  • Link object obtained through Universal channel is valid while the object is alive.
  • Link object obtained through Local channel is valid only while the last local copy of LinkHost is alive.
Template Parameters
TThe type of the object being hosted.
Example of creating and using links:
LinkHost<MyClass> host(new MyClass());
auto link = host.universal_();
if (link) link->doSomething_();

template<typename T >
template<typename U >

void npi::LinkHost< T >::assign_ ( const LinkHost< U > & other)
inlineprivate

Internal helper to assign from another polymorphically related LinkHost object.

Template Parameters
UThe type of object to assign from; must be polymorphically related to T.
Parameters
otherThe LinkHost object to copy from.
Exceptions
std::bad_castif a safe downcast from U* to T* fails.
Precondition
U must be polymorphically related to T.
Note
Private function for internal use only.
  1. If identical, return.
  2. If other.get_() non-null:
    • Upcast via static_pointer_cast if U->T (i.e. if U is T or derived from T).
    • Otherwise downcast via dynamic_pointer_cast, throwing std::bad_cast on failure.
    • Copy uni_ and loc_.
    • Update weakUni_ if applicable.
  3. Else reset obj_, uni_, and loc_ to empty (using reset_()).
