Detailed Description

Header file for custom Qt properties.

This file contains the definition of custom Qt properties for use in the application. These properties are used to manage the state and behavior of various QObject based subclasses in the application.

Note
  • All properties are defined inside npi::props namespace.
  • All properties are const char* const to ensure they are immutable and can be used inside QObject::setProperty() and QObject::property() methods.
